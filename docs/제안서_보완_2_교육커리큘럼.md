# (2) 교육 커리큘럼 상세

## 위치: III. 관리 및 지원 > 3. 교육지원 계획 > [표 III-5] 교육 계획

현재 제안서에는 교육이 간략히 기술되어 있음:
- 관리자 포털 교육 (IT 개발자 2~3명, 4시간)
- 일반사용자 교육 2회
- 관리자 교육 2회

아래는 **상세 커리큘럼**으로 확장한 내용임.

---

## A. 시스템 관리자 교육 (IT 운영팀, 8시간)

### 오전 (4시간) — 인프라 및 운영

| 시간 | 모듈 | 내용 | 실습 |
|------|------|------|------|
| 09:00~09:30 | A-1. 시스템 아키텍처 개요 | 전체 구성도 (LLM + RAG + VectorDB + OCR), 서버 구성, 네트워크 흐름 | 아키텍처 다이어그램 리뷰 |
| 09:30~10:30 | A-2. Docker 컨테이너 운영 | Docker Compose 구조, 서비스별 컨테이너, 로그 확인, 재시작 절차 | `docker compose ps/logs/restart` 실습 |
| 10:30~11:00 | A-3. GPU 서버 관리 | NVIDIA 드라이버/CUDA 확인, GPU 메모리 모니터링, vLLM 서빙 관리 | `nvidia-smi`, vLLM 상태 확인 |
| 11:00~12:00 | A-4. 장애 대응 | 장애 유형별 대응 시나리오 (LLM 무응답, DB 잠금, OOM, 네트워크 단절), 에스컬레이션 체계 | 장애 시뮬레이션 + 복구 실습 |

### 오후 (4시간) — 데이터 및 보안

| 시간 | 모듈 | 내용 | 실습 |
|------|------|------|------|
| 13:00~14:00 | A-5. 백업/복구 | Milvus 벡터DB 백업, SQLite DB 백업, 문서 원본 백업, 전체 복구 절차 | 백업 스크립트 실행 + 복구 테스트 |
| 14:00~15:00 | A-6. 문서 인제스트 파이프라인 | OCR → 청킹 → 임베딩 → 벡터DB 전체 흐름, 재인제스트, 동기화 엔진 | 문서 업로드 → 인제스트 → 검색 확인 |
| 15:00~16:00 | A-7. 모니터링 | Prometheus 메트릭, 대시보드 활용, 알림 설정, 성능 지표 해석 | Grafana 대시보드 실습 |
| 16:00~17:00 | A-8. 보안 관리 | JWT 인증 체계, RBAC 권한, 가드레일 설정, 감사 로그 확인, PII 탐지 | 감사 로그 조회 + 가드레일 규칙 추가 |

### 교육 자료
- 시스템 관리자 매뉴얼 (설치, 운영, 장애 대응, 백업/복구 절차)
- 개발자용 트러블슈팅 가이드 (장애 유형별 대응 시나리오, 디버깅 절차서)
- Agent 로직 상세 사양서 (Router/Planner/MCP 모듈별 상세 설계 문서)
