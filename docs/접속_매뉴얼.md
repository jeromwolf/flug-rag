## 3. 주요 기능 안내

### 3.1 채팅 (RAG 질의응답)
**목표**: LLM과 RAG 성능 평가

1. 중앙의 텍스트 입력창에 질문 입력
2. 전송 버튼 클릭
3. **AI 답변 확인**:
   - SSE 스트리밍으로 토큰이 실시간 표시
   - 신뢰도 배지 확인 (높음/중간/낮음)
   - 출처 카드에서 참고 문서 확인

**시연 질문 예시**:
- 한국가스공사법에서 정한 가스공사의 주요 사업은 무엇인가요?
- 연차휴가 일수는 근속연수에 따라 어떻게 달라지나요?
- 범위 외 질문: "양자역학에 대해 설명해주세요" (안전 메시지 확인)

### 3.2 문서 관리
**목표**: 문서 처리 기능 평가

1. 좌측 메뉴에서 **"문서 관리"** 선택
2. 현재 인제스트된 문서 목록 확인:
   - 총 **92개 파일**, **10,518 청크** (RAG평가용 문서)
   - 파일명, 업로드 일시, 청크 수 표시
3. **새 문서 업로드** (선택):
   - PDF/HWP/DOCX/XLSX/PPTX/TXT 지원
   - 파일을 업로드 영역에 Drag & Drop
   - 인제스트 진행 상태 실시간 확인

### 3.3 OCR 테스트
**목표**: 문서 파싱 및 OCR 기능 평가

1. 좌측 메뉴에서 **"관리자"** → **"OCR"** 탭 선택
2. 문서 또는 이미지 업로드:
   - 스캔된 PDF, 도면, 이미지 지원
   - Upstage Document Parse 기반
3. 추출된 텍스트 결과 확인:
   - 한국어 표 및 도형 인식 품질 평가

### 3.4 에이전트 빌더
**목표**: 워크플로 자동화 기능 평가

1. 좌측 메뉴에서 **"에이전트 빌더"** 선택
2. React Flow 기반 비주얼 캔버스 확인:
   - 노드를 드래그하여 워크플로 구성
   - LLM 노드, 도구 노드, 조건 노드 연결
   - 프리셋 워크플로 5개 제공
3. 워크플로 실행:
   - 입력 데이터 제공 → 자동 실행 → 결과 확인

### 3.5 모니터링 및 통계
**목표**: 품질 관리 및 운영 기능 평가

1. **품질 대시보드**:
   - 청크 길이 분포, 중복 탐지, 의미 완결성 분석
   - 임베딩 추적: 인제스트 작업별 진행 상태

2. **모니터링**:
   - 총 질문 수, 평균 응답 시간
   - 신뢰도 분포 (높음/중간/낮음 비율)
   - 사용 통계 Excel 내보내기

3. **피드백**:
   - 답변 옆 좋아요/싫어요 아이콘으로 평가
   - 싫어요 선택 시 코멘트 입력 가능

## 4. 유의사항

### 접속 및 보안
- **초기 로그인**: 비밀번호 변경 요청이 나올 수 있음 → 안내 따르면 됨
- **동시 접속**: 각 계정별 독립적 세션 (최대 3명 동시 사용 권장)
- **네트워크**: 응답에 수~10초 소요될 수 있음 (정상)

### 시스템 특성
- **첫 질문**: 모델 로드로 인해 10초 이상 걸릴 수 있음 (이후 1~3초)
- **SSE 스트리밍**: 토큰이 실시간으로 표시됨 (ChatGPT와 동일)
- **신뢰도 점수**: 검색 문서 유사도 기반 자동 계산

### 평가 기준
| 평가 영역 | 벤치마크 결과 | 평가 방식 |
|----------|------------|---------|
| **LLM/RAG** | 60문항 100% | 채팅으로 질문 입력 → 정답 여부 판단 |
| **OCR** | 한국어 문서 특화 | OCR 탭에서 문서 업로드 → 텍스트 추출 품질 평가 |
| **Agent Builder** | 5개 프리셋 제공 | 에이전트 빌더에서 워크플로 실행 → 결과 확인 |
