항목사양

파일 첨부 OCR 채팅창에서 파일 첨부 시 자동 OCR 처리 후 AI가 내용 분석
표/도형 인식 한국어 표, 양식, 도면의 구조화된 데이터 추출
다중 파일 지원 여러 파일을 동시에 첨부하여 비교 분석 가능
레이아웃 분석 Upstage Document Parse 기반 정교한 레이아웃 보존

OCR 사용 방법

  1. 채팅 화면에서 하단의 파일 첨부 아이콘을 클릭합니다.
  2. 스캔된 PDF 또는 이미지 파일을 선택합니다.
  3. Upstage Document Parse를 통해 자동 OCR 처리됩니다.
  4. 추출된 텍스트가 AI 컨텍스트에 반영되어 질문에 답변합니다.
  5. 예시 질문: "이 문서의 주요 내용을 요약해주세요"

역할별 핵심 차이점
- Admin vs Manager: Admin만 사용자 관리, 에이전트 빌더, RAG 품질관리 접근 가능
- 4단계 역할(Admin/Manager/User/Viewer)은 예시이며, 귀사의 보안 정책에 따라 역할별 권한을 변경할 수 있습니다.
