3주요 기능 안내

3.1 채팅 (RAG 질의응답)

LLM과 RAG 성능을 직접 평가할 수 있는 핵심 기능입니다.

  1. 하단 텍스트 입력창에 질문을 입력합니다.
  2. 전송 버튼을 클릭하거나 Enter를 누릅니다.
  3. AI 답변 확인:
    • SSE 스트리밍으로 토큰이 실시간 표시됩니다
    • 신뢰도 배지 확인 (높음 / 중간 / 낮음)
    • 출처 카드에서 참고 문서 및 청크 위치 확인
    • 좋아요/싫어요 버튼으로 답변 품질 피드백 입력 가능
그림 4. 채팅 답변 예시 - 신뢰도 점수, 출처 카드, 스트리밍 응답 포함
3.2 문서 관리

문서 업로드 및 인제스트 현황을 확인할 수 있습니다.

  1. 좌측 메뉴에서 "문서 관리"를 선택합니다.
  2. 현재 인제스트된 문서 목록을 확인합니다:
    • 총 92개 파일, 10,518 청크 (RAG평가용 문서)
    • 파일명, 업로드 일시, 청크 수, 처리 상태 표시
  3. 새 문서 업로드 (선택 사항):
    • PDF, HWP, DOCX, XLSX, PPTX, TXT 지원 (6가지 형식)
    • 파일을 업로드 영역에 Drag & Drop
    • 인제스트 진행 상태 실시간 확인
그림 5. 문서 관리 화면 - 인제스트 현황, 문서 업로드, 처리 상태 확인
3.3 OCR 테스트 (관리자 계정)

Upstage Document Parse 기반 한국어 문서 OCR 기능입니다.

  1. 좌측 메뉴에서 "관리자"를 선택합니다.
  2. 상단 탭에서 "OCR" 탭을 클릭합니다.
  3. 문서 또는 이미지를 업로드합니다:
    • 스캔된 PDF, 도면, 이미지 지원
    • 한국어 표 및 도형 인식 포함
  4. 추출된 텍스트 결과를 확인하고 품질을 평가합니다.
그림 6. 관리자 설정 화면 - OCR, 프롬프트 관리, 모델 설정 탭 포함
3.4 에이전트 빌더

React Flow 기반 비주얼 캔버스로 워크플로를 설계하고 실행합니다.

  1. 좌측 메뉴에서 "에이전트 빌더"를 선택합니다.
  2. 비주얼 캔버스를 확인합니다:
    • 노드를 드래그하여 워크플로 구성
    • LLM 노드, 도구 노드, 조건 분기 노드 연결
    • 프리셋 워크플로 5개 제공 (목록에서 선택)
  3. 워크플로를 실행합니다:
    • 입력 데이터 제공 후 실행 버튼 클릭
    • 각 노드의 실행 상태가 실시간으로 표시됨
    • 최종 결과 확인
그림 7. 에이전트 빌더 - 비주얼 DAG 캔버스, 노드 연결, 프리셋 워크플로
그림 8. 에이전트 워크플로우 실행 결과 - 도구 체이닝 및 단계별 실행 상태
3.5 모니터링 및 품질 관리 (관리자/매니저 계정)

운영 현황 및 RAG 품질 지표를 대시보드에서 확인합니다.

모니터링 대시보드

총 질문 수, 평균 응답 시간, 신뢰도 분포 (높음/중간/낮음) 비율, 사용 통계 Excel 내보내기

RAG 품질 대시보드

청크 길이 분포, 중복 탐지, 의미 완결성 분석, 임베딩 추적 (인제스트 작업별 상태)

피드백 관리

사용자 좋아요/싫어요 데이터 집계, 코멘트 조회, 개선 사항 추적

접근 로그

사용자별 쿼리 이력, 접속 로그, 이상 패턴 탐지 현황

그림 9. 모니터링 대시보드 - 사용 통계, 응답 시간, 신뢰도 분포
그림 10. RAG 품질 관리 - 청크 분포, 임베딩 추적, 중복 탐지

6자주 묻는 질문 (FAQ)

Q1. 로그인이 되지 않습니다.
계정명과 비밀번호를 다시 확인하세요 (영문 대소문자 구분). 브라우저 쿠키를 삭제(설정 > 개인정보 > 쿠키 삭제) 후 재시도하시거나 다른 브라우저(Chrome, Edge)에서 접속해 보세요.
Q2. 채팅 응답이 나오지 않거나 로딩이 멈춥니다.
네트워크 연결을 확인하세요. 첫 번째 질문은 모델 로드로 인해 최대 30초가 소요될 수 있습니다. 그 이후에도 응답이 없으면 페이지를 새로고침(F5) 후 재시도하세요.
Q3. 문서 업로드는 어떤 형식을 지원하나요?
PDF, HWP, DOCX, XLSX, PPTX, TXT 6가지 형식을 지원합니다. 스캔 문서의 경우 OCR 탭을 통해 처리할 수 있습니다.
Q4. 동시에 여러 명이 접속할 수 있나요?
가능합니다. KGT-G01 ~ KGT-G10 계정이 각각 독립적으로 동작하므로 10명이 동시에 접속할 수 있습니다.
Q5. 에이전트 빌더 메뉴가 보이지 않습니다.
에이전트 빌더는 모든 권한에서 접근 가능합니다. 좌측 사이드바에서 "에이전트 빌더" 항목을 찾아보세요. 사이드바가 접혀 있는 경우 상단 메뉴 아이콘을 클릭하여 펼치면 됩니다.
Q6. 신뢰도 점수는 어떻게 산출되나요?
검색된 문서 청크와 질문의 벡터 유사도를 기반으로 자동 계산됩니다. 0.7 이상: 높음(녹색), 0.4~0.7: 중간(노란색), 0.4 미만: 낮음(빨간색)으로 표시됩니다.
